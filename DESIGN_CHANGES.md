# Design update

## 変更した構成

- 入力画像の右側を「推定ライト配置」専用の領域に変更
- 画像解析結果を、入力画像とライト配置の下へ移動
- 制作ガイドとデータ書き出しを下層へ整理
- アップロードを初期表示にし、生成機能は切り替え式で維持

## デザイン方針

- 明るいニュートラル背景と白いカードで情報のまとまりを明確化
- 深いグリーンをアクセントにし、解析ツールらしい落ち着いた印象へ統一
- 見出し、補足、数値、操作の優先順位が一目で分かる文字設計
- ボタン、アップローダー、タブ、メトリクスの形状と余白を統一
- 横長画面では「入力画像 → ライト配置」の比較を最優先

## 起動

```powershell
pip install -r requirements.txt
streamlit run app.py
```

## app2.py：レスポンシブ・ワンビュー版

- 画面の最大幅制限をなくし、iMacを含む横長画面で左右の余白を抑制
- 外側の余白を画面幅に応じて可変化
- 入力完了後は入力UIを折りたたみ、結果表示を画面上部へ移動
- 入力画像・照明配置図・解析結果を3カラムで同時表示
- 画像と3D配置図の高さを画面サイズに合わせて抑制
- 狭い画面では各パネルが縦方向へ切り替わるレスポンシブ構成
- 詳細解析・制作ガイド・書き出しは折りたたみ領域へ整理

起動時は次のファイルを指定します。

```powershell
streamlit run app2.py
```
