# イメージ外側構築ツール

AI生成画像の「存在しない外側」を、物理セット制作の条件として推定するStreamlit MVPです。

## 起動

```powershell
.\.venv\Scripts\python.exe -m streamlit run app.py
```

初回のみ依存関係を入れる場合:

```powershell
pip install -r requirements.txt
```

## できること

- AI生成画像をアップロード
- 明るさ、暗部比率、明部比率、明暗分布を解析
- 明るい領域の塊を複数検出し、キーライト、フィルライト、リム/バックライト候補として分類
- エッジ、彩度、画面内位置から疑似深度マップを生成
- 明るさマップと疑似深度マップを表示
- 解析結果からライトの距離、高さ、角度、強さを推定
- 複数光源がある場合は、それぞれの距離、高さ、角度、強さを個別に推定
- ライト・被写体・カメラを3D配置ビューで表示
- 3D配置ビューをマウスドラッグで回転、ホイールで拡大縮小
- ライトとカメラの位置を上面図、前面図、側面図で表示
- セット制作ガイドをJSONまたはMarkdownで書き出し

## 推定の考え方

ライト推定は、以下の値を使って計算します。

- 平均明度
- 明度の p10 / p50 / p90
- 暗部比率
- 明部比率
- 明るい領域の重心
- 明るい領域の個数、面積、ピーク明度
- 疑似深度の近景量
- 疑似深度の複雑度

疑似深度は、MiDaSのような学習済み単眼深度モデルではなく、ローカルで軽く動くルールベースの近似です。画面下側ほど近い、エッジや局所コントラストが強いほど近い、彩度が強い領域ほど前景に出やすい、という複数の手がかりを合成しています。

## 構造

```text
AI画像
  ↓
画像解析
  - 明るさマップ
  - 暗部 / 明部比率
  - エッジ量
  - 疑似深度マップ
  - 色パレット
  ↓
Physical Inference Engine
  - 光源位置の推定
  - ライト強度の計算
  - 距離 / 高さ / 角度の計算
  ↓
Scene Compiler
  - ライト配置
  - 物体配置
  - カメラ配置
  - フレーム外の制作手順
  ↓
制作ガイド
```
